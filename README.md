# JavaScript Logic & Basic Exercises

โปรเจกต์นี้รวบรวมไฟล์แบบฝึกหัดภาษา JavaScript พื้นฐาน เพื่อศึกษาโครงสร้างข้อมูล, อัลกอริทึม, และการใช้งาน Callback Functions

## 📝 รายละเอียดโปรเจกต์ (Description)
ชุดโค้ดนี้ประกอบไปด้วยการแก้โจทย์ปัญหาทางคอมพิวเตอร์เบื้องต้น เช่น:
- การคำนวณเกรดจาก Array (`let arr.js`)
- การหาเลข Prime Number (`prime number.js`)
- การใช้งาน Callback Function พื้นฐาน (`callback function.js`)
- การจัดการ Object และ Array ใน JavaScript
